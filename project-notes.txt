# PROJECT NOTES #

Weather App
- can display 5-day weather forecast 
- city name
- current weather icon
- temperature
- humidity
- wind speed
- recording of both high and low temperatures of each day
- images for sunny/rainy/cloudy/snowy weather conditions
- refresh every five minutes with the exact temperature and weather conditions

Process
User searches a city name
---------
Utilize geocoding API (https://openweathermap.org/api/geocoding-api) to obtain latitude + longitude of city
---------
Call current weather data from OpenWeather (https://openweathermap.org/current) using latitude + longitude details
---------
Display 5 day weather forecast for city
